# Online Exam Malpractice Detection System

## 📌 Overview

The Online Exam Malpractice Detection System is an Artificial Intelligence based application designed to monitor students during online examinations and detect suspicious activities.

This system uses computer vision techniques to observe student behavior through a webcam and identify possible cheating activities such as multiple face detection, unusual movements, or absence from the screen.

The aim of this project is to improve the integrity and fairness of online examinations.

---

## 🚀 Features

* Real-time webcam monitoring
* Face detection using Computer Vision
* Multiple face detection (possible cheating)
* Suspicious activity alerts
* Automated monitoring during exams
* Simple and easy-to-use interface

---

## 🛠 Technologies Used

* Python
* OpenCV
* NumPy
* Machine Learning
* Computer Vision

---

## ⚙️ How the System Works

1. The system activates the student's webcam.
2. Computer Vision algorithms detect the student's face.
3. If multiple faces are detected, the system flags it as suspicious activity.
4. If the student moves away from the screen for a long period, the system can detect it.
5. Suspicious activities can be logged or reported to the examiner.
